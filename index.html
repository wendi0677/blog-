繼續閱讀

痛苦會過去，經驗會留下，不要害怕挑戰

2022 年 一月卡斯伯

生物相關科系畢業，進入職場後先在服務業任職六年，參加「Vue 作品實戰班」後轉職成為前端工程師。今天我們就來聽聽看海獺是如何成功跨領域轉職的吧！

轉職的決心

海獺在轉職前從事服務業，同時也是兩歲小孩的母親，面對家庭責任和孩子的照顧，想要轉換職業並不容易。然而，海獺得到了先生的支持，他是一位後端工程師，對前端開發有一定了解，不僅支持海獺的決定，還給予了她寶貴的學習建議。
在做出全職學習的決定後，海獺首先購買了 Vue 3 實戰影音課程。剛好在購課後，她得知了 Vue 作品實戰班的開班消息，於是決定加入，逼自己一把。

學習歷程：有效運用時間，兼顧家庭及學習

因為先生是後端，海獺跟大部分人的學習路線不一樣，從框架開始逆學回去，Vue 實戰影音課程＞Vue 作品實戰班（也利用影音課程，學習 JS＞CSS），完成直播班課程後，持續尋找各種可用資源來學習，例如佛心網友的教學課程。
雖然是全職學習，但由於需要照顧小孩，她平日僅能投入 5 到 6 小時於學習，而假日則將時間留給家庭。在這樣的情況下，她必須有效地管理時間，以下是海獺分享的學習方式：

海獺的學習方式：
影音課程自學一個月後，就直上直播班，遇到問題時就會回去看影音課程，觀看順序：常用 JS 觀念 > 快速入門 Vue.js > 指令語法介紹 > Options API > Vue Router > Composition API > Vite > 元件 > 進階 API ，直播班最終任務有完成 90%，完成了人生的第一個網站作品。

一開始嘗試同時應對每日任務和主線任務，但發現有些力不從心。因此，後來將重點放在主線任務上，只有在需要時才會回去處理每日任務，這樣更有效率地利用了她有限的學習時間。

同時海獺也會找其他學習資源，像是 The F2E 前端 & UI 修煉精神時光屋等前端活動，練習做 Side Project；也會運用各種社群資源、Google、ChatGPT；六角的 Discord 社群也會盡量發問，都有助教協助回覆，如果害羞，也會趁直播班交作業時提問助教。
海獺也分享，意識到記憶不可能永遠保持清晰，因此培養了做筆記的習慣，她從八月開始參加鐵人賽，有助於快速回顧所學知識，對海獺來說是一個具有挑戰性的過程，完賽帶來的成就感激勵了她，意識到真正的學習是在輸出的過程中發生的，因此她將寫作當作一種筆記方式，進一步鞏固了所學的知識。

海獺：六角的資源很多，用就對了！

一開始選擇六角課程，主要是考慮到經濟狀況和小孩成長，選擇能以線上課程為主，也比較有時間陪伴小孩。
六角的課程中，海獺最喜歡直播課了，喜歡聽兩位老師分享知識點或小撇步，遇到不理解的問題老師都可以直接回答，互動性很高。還有助教批改的服務，學習時，我們通常會用自己的認知出發，但如果有助教可以從不同角度幫你改 code 可以幫你脫離盲區。此外，六角的網路資源豐富，Discord 社群幫助很大，即使不敢發問，也能看其他同學的問題，學習到不同的知識點。
最後還有校長協助履歷健檢，在出社會一陣子後，面對不同專業的求職，校長給了很多建議、信心。

求職歷程

從開始學習到拿到 offer ，大約花了一年的時間，總共丟了約 170 家履歷，不局限於使用 104、1111 等常見求職平台，Yourator、Cake Resume、 LinkedIn…等也都多方嘗試。
面對跨領域的面試，海獺描述了初期的挑戰：腦袋一片空白。這對許多人來說是一個常見的情況，但後續逐漸修正問題，加強練習，有空時多寫多看，面試前多看網友的面試心得，多做一點準備，就會慢慢熟練！
海獺也建議，如果有人願意協助模擬面試，非常建議參加！能夠發現自己的盲點並加以改善。這種實際的演練不僅提高了自信心，也使能更加熟悉面試的過程和要求，進一步增強了應對面試的能力。

給學弟妹的話

海獺建議學弟妹：在直播班分組時，可以勇敢挑戰擔任「組長」，做一些以前沒做過的事。在直播課程結束後，可以找志同道合的組員、網友，一起為了共同目標而舉辦讀書會，或是線上線下活動等等。
學習過程中遇到挫折是不可避免的，但重要的是要學會調整心態，讓自己沈浸在學習中。像是自己有時候會解不出 code，懊惱很久，但後來發現只是錯了一小行語法，解出來後豁然開朗，成就感很高。

海獺也想和學弟妹分享這句話：「痛苦會過去，經驗會留下」。想要轉職的人必須下定決心，願意投入心力和時間，你的動機應該都是為了追求更好的未來，無論是為了金錢還是其他目標，都不要害怕挑戰；尤其是在前端領域，語法和技術更新迅速，轉職者必須敢於迎接這些變化。

海獺也很喜歡卡斯伯的這句話，推薦給學弟妹：
「堅持不懈是成功的關鍵，跨越所有障礙，終能實現理想的目標。」

關於我

接列的老論看我那就同事。覺得真的特但其實，能一間真的感覺這是並不應該在幹嘛，不得不得看他們吧排球的東西。
